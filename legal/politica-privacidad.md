# Política de privacidad de ReelSwapAI

Última actualización: mayo de 2026

ReelSwapAI es una aplicación que permite generar imágenes y vídeos con intercambio de rostro mediante inteligencia artificial, usando fotos y vídeos seleccionados por el usuario.

## Contacto

Para cualquier consulta sobre esta política de privacidad, puedes escribir a:

reelswapai.app@gmail.com

## Datos que podemos tratar

ReelSwapAI puede tratar la siguiente información:

- Datos de cuenta, como el correo electrónico usado para registrarse.
- Fotos y vídeos seleccionados por el usuario para generar resultados.
- Imágenes o vídeos generados.
- Saldo de tokens, historial de compras e historial de generaciones.
- Información técnica necesaria para el funcionamiento de la app.

## Para qué usamos los datos

Usamos los datos para:

- Proporcionar el servicio de generación con IA.
- Gestionar cuentas de usuario y saldo de tokens.
- Procesar compras y validar información relacionada con pagos.
- Mejorar la estabilidad y funcionamiento de la app.
- Prevenir usos indebidos, fraude o actividad no autorizada.

## Fotos, vídeos y procesamiento con IA

Las fotos y vídeos seleccionados por el usuario se suben únicamente cuando son necesarios para generar el resultado solicitado.

Los archivos pueden ser procesados por servicios externos necesarios para ofrecer la generación con IA y el manejo temporal de archivos.

Los resultados generados pueden almacenarse temporalmente para permitir al usuario guardarlos o compartirlos. Siempre que sea posible, evitamos conservar archivos multimedia durante más tiempo del necesario.

## Servicios de terceros

ReelSwapAI utiliza servicios de terceros que pueden tratar datos como parte del funcionamiento de la app, incluyendo:

- Firebase, para autenticación, base de datos y almacenamiento de datos de usuario.
- RevenueCat, para la gestión de compras dentro de la app.
- Apple App Store, para pagos y compras dentro de la app.
- Cloudinary, para gestión temporal de archivos multimedia.
- Proveedores de IA, para procesar las generaciones de intercambio de rostro.

Estos servicios tratan los datos según sus propias políticas de privacidad y prácticas de seguridad.

## Compras y pagos

Las compras dentro de la app se procesan a través de Apple App Store. ReelSwapAI no recopila ni almacena directamente datos de tarjetas bancarias.

La información relacionada con compras puede ser tratada por Apple y RevenueCat para validar pagos y actualizar el saldo de tokens.

## Responsabilidad y consentimiento del usuario

El usuario debe subir únicamente imágenes y vídeos propios o para los que tenga permiso.

No está permitido usar ReelSwapAI para suplantar, engañar, acosar, explotar o crear contenido con la imagen de otra persona sin su consentimiento.

## Conservación de datos

Podemos conservar datos de cuenta, saldo de tokens, historial de compras e historial de generaciones mientras sean necesarios para prestar el servicio, mantener la seguridad y cumplir obligaciones legales.

Los archivos multimedia temporales pueden eliminarse después del procesamiento o después de que el usuario guarde o comparta el resultado.

## Seguridad

Aplicamos medidas razonables para proteger los datos del usuario. Sin embargo, ningún sistema puede garantizar una seguridad absoluta.

## Menores

ReelSwapAI no está dirigida a menores. El usuario debe cumplir la edad mínima exigida por la legislación aplicable y por las normas de la plataforma.

## Derechos del usuario

Según tu ubicación, puedes tener derecho a acceder, rectificar, eliminar o limitar el tratamiento de tus datos personales.

Para ejercer estos derechos, puedes escribir a:

reelswapai.app@gmail.com

## Cambios en esta política

Podemos actualizar esta política de privacidad cuando sea necesario. Cualquier cambio se publicará con una nueva fecha de actualización.
